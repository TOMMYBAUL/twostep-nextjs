🛠 Setup technique global

①

Lenis — smooth scroll global. npm install lenis. 4 Ko. Initialisation dans le layout. Tout le scroll de la homepage devient fluide et "buttery".

②

Framer Motion — déjà installé (motion v12). useInView pour les animations d'entrée, useScroll + useTransform pour le parallax et le sticky.

③

prefers-reduced-motion — toutes les animations désactivées si l'utilisateur le demande. Non négociable.

④

Spring par défaut — { type: "spring", stiffness: 200, damping: 30 } pour tout sauf les compteurs (ease-out). Défini une seule fois, réutilisé partout.
