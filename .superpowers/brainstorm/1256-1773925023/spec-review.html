S1 — Authentification

S1.1 — Connexion /auth/login

  • Formulaire email + mot de passe
  • Lien "Pas encore de compte ?" → signup
  • Lien "Mot de passe oublié" → Supabase reset flow
  • Succès → redirect /dashboard
  • Déjà connecté → redirect /dashboard

S1.2 — Inscription /auth/signup

1
Compte
Email + mot de passe + confirmation. Min 8 caractères.
2
SIRET
14 chiffres → vérification API INSEE via POST /api/verify-siret. Pré-remplit le profil.
3
Boutique
Nom, adresse, téléphone, géoloc. Pré-rempli depuis INSEE.
