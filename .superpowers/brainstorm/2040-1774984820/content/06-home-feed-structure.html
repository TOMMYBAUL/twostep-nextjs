Onglet Accueil — Structure du feed

Comment organiser les sections quand l'utilisateur ouvre l'app ?

A

Promos hero → Tendances → Pour toi / Suivis

Les promos sont le premier truc qu'on voit (carrousel grande taille). Ensuite les tendances. Puis le feed infini avec toggle Pour toi/Suivis. Les promos accrochent, le feed retient.

📱 Scroll : Promos 🔥 (carrousel) → Tendances (grille) → [Pour toi | Suivis] (feed infini)
B

Pour toi / Suivis en haut → Promos intégrées dans le feed

Le toggle Pour toi/Suivis est tout en haut (comme Instagram). Les promos apparaissent comme des cartes mises en avant dans le feed (fond orange, badge 🔥). Pas de section séparée.

📱 Scroll : [Pour toi | Suivis] → feed mixte (produits + promos intercalées + tendances)
C

Stories → Promos → Tendances → Pour toi / Suivis

En tout premier, les stories des boutiques suivies (bulles en haut comme Instagram/Snapchat). Puis promos, tendances, feed. Les stories créent un lien émotionnel avec les boutiques.

📱 Scroll : Stories (bulles) → Promos 🔥 → Tendances → [Pour toi | Suivis]
D

Comme le Discover actuel (ce qu'on a déjà)

On garde la structure actuelle : header Two-Step → filtres catégorie/taille → toggle Pour toi/Suivis → sections (promos, tendances, boutique à découvrir, tout près de toi). Ça marche, on améliore juste le design.

📱 Scroll : Filtres → [Pour toi | Suivis] → Promos → Tendances → Boutiques → Tout près de toi

Ma recommandation : Option A. Les promos en hero captent l'attention immédiatement (raison de revenir). Puis les tendances pour le côté "qu'est-ce qui est populaire". Puis le feed infini Pour toi/Suivis pour garder les gens. Vu qu'on a un onglet Promos dédié, la section hero ici sert de teaser — "voir tout →" renvoie vers l'onglet Promos.
